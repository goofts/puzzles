Problem Statement

You are given the following data:

  • ints a, b, c, d, e, and f: positive integer coefficients.
  • longs n and m: upper bounds for a sum and a product.
  • int[] mat: the elements of a K*K matrix (in row-major order)

Let T(p) denote the p-th power of the matrix described by mat.

We can now define a new K*K matrix as follows:



Find this matrix. Compute and return the sum of all its elements, modulo 2,147,483,648 . Note that the product is inside the sum.

  • Class Tribonacci

  • Returns int

(be sure your method is public)

Limits

  • Time limit (s) 8.000
  • Memory limit (MB) 256

Notes

  • The value of K is not given explicitly, because it's uniquely determined by mat.

Constraints

  • Each of the numbers a, b, c, d, e, and f will be between 1 and 10^9, inclusive.
  • n will be between 1 and 10^18, inclusive.
  • m will be between 1 and 10^18, inclusive.
  • K will be between 1 and 50, inclusive.
  • mat will contain exactly K elements, and each element of mat will contain exactly K values.
  • Each number in mat will be between 1 and 10^9, inclusive.
